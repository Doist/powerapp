Github Integration Setup

To complete the Github integration setup, you need to configure your github repository webhook setting.

  1. In your Github repository setting, go to the "Webhooks & Services" section. Create a new webhook.
  2. Fill in the "Payload URL" field with the following URL:
    {{ extra_context.webhook_url }}
  3. For the webhook event setting, select "Issues", and complete the setup by saving the setting.
